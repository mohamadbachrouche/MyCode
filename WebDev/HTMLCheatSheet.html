Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

HTML Elements

HTML documents are made up of elements, which are represented by tags. Here are some common HTML elements:

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Links
  • <img>: Images
  • <ul>, <ol>, <li>: Lists
  • <div>: Division (block element)
  • <span>: Inline container
